# 超簡単！5分で確認する方法

## 🎯 まずはこれだけ試してください

### ステップ0: 依存関係をインストール（初回のみ・必須）

**重要**: 最初に必ずこれを実行してください！

1. **ターミナルを開く**
   - Macの「ターミナル」アプリを開く

2. **プロジェクトフォルダに移動**
   ```bash
   cd /Users/masakinakata/LanguageMatching
   ```

3. **依存関係をインストール**
   ```bash
   npm install
   ```
   
   ⏰ **数分かかります**（初回のみ）
   - 成功すると、`node_modules` フォルダが作成されます
   - エラーが出なければOK

### ステップ1: サーバーが起動するか確認（2分）

1. **以下のコマンドをコピー＆ペースト**
   ```bash
   npm run dev
   ```

3. **成功メッセージを確認**
   ```
   🚀 Server running on port 3000
   ```
   
   このメッセージが表示されれば、**サーバーは正常に起動しています！** ✅

4. **ブラウザで確認**
   - ブラウザで以下のURLを開く:
   ```
   http://localhost:3000/health
   ```
   - `{"status":"ok"}` が表示されればOK ✅

**これで基本確認は完了！**

## 📱 WhatsAppで確認する方法

### 重要: まず準備が必要です

WhatsAppで確認するには、以下が必要です：

1. **Twilioアカウント**（まだの場合）
   - https://www.twilio.com でアカウント作成
   - 無料トライアルでOK

2. **環境変数の設定**
   - `.env`ファイルにTwilioの認証情報を設定

3. **ngrok（ローカル環境の場合）**
   - ローカルでテストする場合、ngrokが必要
   - https://ngrok.com からダウンロード

### 簡単な確認方法（推奨）

**まずはサーバーが起動するか確認するだけでもOKです！**

WhatsAppでの確認は、環境が整ってからでも大丈夫です。

## 🆘 エラーが出た場合

### エラー1: `npm: command not found`

**解決方法**: Node.jsをインストール
- https://nodejs.org からダウンロード

### エラー2: `Cannot find module` または `Cannot find package 'express'`

**解決方法**: 依存関係をインストール（**これが最も多いエラーです！**）
```bash
cd /Users/masakinakata/LanguageMatching
npm install
```

**重要**: このエラーが出たら、必ず `npm install` を実行してください！

### エラー3: ポート3000が使われている

**解決方法**: 別のポートを使う
- `.env`ファイルで `PORT=3001` に変更
